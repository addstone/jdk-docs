Java 用 JavaBeansTM コンポーネン ト API

JavaBeans は、 Java の "write once - run everywhere" という機能を、再利用可能なコンポーネント開発のために拡張します。実際、コードはすべての OS 上およびすべてのアプリケーションの内部環境で実行できるという機能に対して、JavaBeans は相互利用を重要なステップとしてとらえています。Beans 開発者は、従来のプラットフォームを使用する顧客を失うことなしに、ネットワークソフトウェア市場においても顧客に対するサポートを保証します。 それは、JavaBeans が ActiveX、OpenDoc、および LiveConnect と相互操作するからです。

Beans 開発者は、次にリストされるBeans 開発キットをダウンロードする必要もあります。
