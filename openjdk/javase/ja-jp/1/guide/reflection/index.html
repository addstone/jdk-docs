リフレクション

ロードしたクラスのフィールド、メソッド、および構築子に関する情報の参照と、参照したフィールド、メソッド、および構築子を使ってセキュリティ制限の範囲内でオブジェクトの基本部分の操作を、Java コードで可能にします。この API により、 (実行時のクラスに基づく) ターゲットオブジェクトの public メンバまたは所定クラスが宣言したメンバへのアクセスを必要とするアプリケーションを作ることが可能となります。
