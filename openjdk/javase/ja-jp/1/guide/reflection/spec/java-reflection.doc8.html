クラス java.lang.reflect.InvocationTargetException 

    package java.lang.reflect;

    public class InvocationTargetException extends Exception
InvocationTargetException は、呼び出されたメソッドまたは構築子がスローした例外をラップする確認済みの例外です。

構築子

InvocationTargetException

    public InvocationTargetException(Throwable target)
指定の target 例外を用いて、InvocationTargetException を構築します。

InvocationTargetException

    public InvocationTargetException(Throwable target, String detail)
指定の target 例外および例外を記述する detail メッセージ String を用いて、InvocationTargetException を構築します。

メソッド

getTargetException

    public Throwable getTargetException()
InvocationTargetException がラップした、基本ターゲット例外を返します。
