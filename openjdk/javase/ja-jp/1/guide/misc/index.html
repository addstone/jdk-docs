その他の機能

  • Byte クラス、Short クラス、Voidクラス
    新しいクラス Byte および Short が追加され、byte 型と short 型がラップされた数値として組み込まれました。抽象クラスの Number には、2 つの新しいメソッド byteValue と shortValue が加わりました。これらのメソッドのデフォルトの実装では intValue メソッドを使用します。また、新しい Void クラスも追加されました。これはインスタンス化することのできない場所を確保するためだけのクラスです。
  • @deprecated タグ
    ドキュメトのコメント内で、クラスやメンバに「推奨されていない」ことを表すマークをつ けるための機能です。開発者に新しいクラスやメンバを使うように指示します。
  • リソースファイルへのアクセス
    リソースの位置に依存せずに、リソースを見つけるメカニズムを提供する API です。こ のメ カニズムを使用して、複数の HTTP 接続を使用してネットワーク上からロードされたアプレ ットや、JAR ファイルを使用してロードされたアプレット、CLASSPATH を使用してインスト ールされた「ライブラリ」などから、リソースファイルを見つけることができます。
  • APPLET タグ
    HTML で定義されている <APPLET> タグの機能拡張です。アプレットビューアまた は Web ブラウザにアプレットをロードできます。
