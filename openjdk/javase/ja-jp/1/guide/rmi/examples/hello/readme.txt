JDK 1.1Beta リリースの docs/guide/rmi/examples/hello ディレクトリに、
RMI チュートリアルに含まれる RMI を使った "Hello World!" プログラムの
ソースコードがあります。 JDK 1.1 Beta リリースをダウンロードした後、イ
ンストラクションにしたがってゼロからプログラムを作って実行させるのが面
倒ならば、スクリプトを実行してプログラムのビルドと実行をさせることもで
きます。実行中にコメントが表示されます。 

1.) JDK 1.1 Beta のインストレーションディレクトリ 
    /docs/guide/rmi/examples/hello へ移動してください。 

2.) Solaris の場合は run スクリプトを実行します。 スクリプトはレジスト
    リが現在実行中ではないことを仮定しています。 

3.) Windows システムの場合は run.bat を実行します。 実行が完了しました
    ら、レジストリとサーバープロセスのために作られたウィンドウを消して
    ください。
