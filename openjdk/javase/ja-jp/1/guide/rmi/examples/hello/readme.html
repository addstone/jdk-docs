RMI を使った Hello World プログラム

JDK 1.1Beta リリースの docs/guide/rmi/examples/hello ディレクトリに、RMI チュートリアルに含まれる RMI を使った "Hello World!" プログラムのソースコードがあります。 JDK 1.1 Beta リリースをダウンロードした後、インストラクションにしたがってゼロからプログラムを作って実行させるのが面倒ならば、スクリプトを実行してプログラムのビルドと実行をさせることもできます。実行中にコメントが表示されます。

  1. JDK 1.1 Beta のインストレーションディレクトリ /docs/guide/rmi/examples/hello へ移動してください。

  2. Solaris の場合は run スクリプトを実行します。 スクリプトはレジストリが現在実行中ではないことを仮定しています。

  3. Windows システムの場合は run.bat を実行します。 実行が完了しましたら、レジストリとサーバープロセスのために作られたウィンドウを消してください。
