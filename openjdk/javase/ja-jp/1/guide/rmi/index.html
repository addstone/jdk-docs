RMI - リモートメソッド呼び出し (Remote Method Invocation)

RMI の使用によりプログラマは分散した Java-to-Java アプリケーションを作成することがでるようになり、リモートにある Java オブジェクトのメソッドを別な Java の仮想マシン、別なホストの Java 仮想マシンから呼び出すことができるようになります。 Java プログラムは、リモートオブジェクトへの参照を得てからリモートオブジェクトへの呼び出しが可能になります。リモートオブジェクトへの参照を得る方法としては、RMI が提供するブートストラップ名前サービスからルックアップによるか、引数または返り値として得ることができます。クライアントはサーバのリモートオブジェクトを呼び出すことができますし、サーバは他のリモートオブジェクトに対してのクライアントにもなり得ます。 RMI はオブジェクト直列化機能を使ってパラメータの整列、再整列を実行しますから型の切り詰めは行わず、真のオブジェクト指向ポリモーフィズムをサポートします。
RMI チュートリアル
  • はじめましょう
    RMI チュートリアル、はおなじみの Hello World プログラムの分散システム版を Java RMI (Remote Method Invocation) を使って作成する手順を順を追って示してくれます。 Hello World アプレットはサーバへのリモートメソッド呼び出しを行い、そこから自分をダウンロードしてメッセージ "Hello World!"を取り出します。
