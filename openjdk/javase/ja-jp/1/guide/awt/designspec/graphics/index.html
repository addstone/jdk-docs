AWT 1.1 グラフィックスとイメージの拡張

最終更新: 1996 年 10 月 30 日

JDK の 1.1 バージョンでは、多くのグラフィックスとイメージングの拡張が提供されて います。次のリンクで、この問題を詳述するドキュメンテーションと問題解決のために開発された新しい API に到達します。

  1. グラフィックスクリップをセットおよびリセットする、新しいメソッド。
  2. イメージを即座に同時にクロッピング、スケーリング、フリッピングできる、新しいイメージレンダリングメソッド。
  3. 既存の実装との混乱を生じたいくつかの問題に取り組むために作成された、新しいイメージスケーリング動作と機構。
  4. MemoryImageSource に追加された、新しいアニメーション機能。
  5. 開発者共同体からのニーズに答えるために追加された、新しい PixelGrabber メソッド。
