JNI - Java ネイティブインタフェース

Java ネイティブインタフェースは、Java ネイティブメソッドを書いたり、 Java仮想マシンを ネイティブアプリケーションに組み込んだりするための標準プログラミング インタフェースです。 主な目的は、所定のプラットフォーム上のすべての Java 仮想マシンの実装間のネイティブメソッドライブラリのバイナリ互換性にあ ります。
