JDBCTM - Java と データベースの接 続

Java Database Connectivity は、さまざまなリレーショナルデータベースへの共通のアクセスができる、標準のSQL データベースアクセスインタフェースです。 Java Database Connectivityは、高レベルのツールとインタフェースを構築できる共通のベースも提供しています。このプログラムは「ODBCブリッジ」に付いています。. ブリッジは、ODBC標準の C API についての JDBC を実装したライブラリです。
