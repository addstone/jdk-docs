JDK 1.1.1 クラスリスト

以下は、JDK 1.1.1 の java.* パッケージに含まれる、すべての public クラスの数です。クラスは一切 1.1 から 1.1.1 の間の実装で追加されたり、削除されたりしていません。このリストは、javadoc で生成した HTML ドキュメントをまとめたものです。 含まれるクラス数は次のとおりです:

JDK 1.0.2 JDK 1.1.1
Public java.* パッケージ 8 22
Public java.* クラスおよびインタフェース 211 477

この数は 26 個の awt.peer クラスおよびピアパッケージがドキュメントされていないため、JDK 1.1 より少なくなっています。

(リストの生成に使用したプログラム: /home/dkramer/bin/diffclassnames.sh)

+プラス記号は、JDK 1.0.2 と比較した JDK 1.1 用の新しいパッケージ、クラス、またはインタフェースを意味します。
注意: プラス記号のないクラスでも、JDK 1.1 用にメソッドを追加したものもあります。

注意 - 以下のパッケージ内の API は、直接呼び出さないでください:
   java.text.resources.*
このパッケージは一時的なものなので、このリストには入っていません。
次回の API 変更リリースで修正する予定の1.1の問題を解決するため、 このパッケージ内の API を public にする必要がありました。 java.text.resources のパッケージ内の API および実装は変更され、 API が新しく追加されたり削除される可能性があります。
これらは private パッケージとして扱ってください。
