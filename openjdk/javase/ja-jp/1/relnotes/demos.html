アプレットとアプリケーションのデモンストレーション

ブラウザの要件

一般的に、アプレットは AppletViewer または以下に述べる Java 互換ブラウザで実行可能です。もちろん、新しい Java 1.1 API に依存するアプレットは Java 1.0 だけをサポートするブラウザでは動作しないので、バージョンの制限はあります。
  • Java 1.0 アプレット - Netscape 3.0、Internet Explorer 3.0、 HotJava プリベータ、AppletViewer 1.0 またはこれら以降のすべてのバージョンを使用
  • Java 1.1 アプレット - HotJava 1.0、AppletViewer 1.1 または Netscape か Internet Explorer の今後出る 1.1 互換バージョンを使用
AppletViewer は JDK 1.1 ソフトウェアのリリースに添付して提供されます。

一般的に、JDK 1.0.2 で実行できるすべてのアプレットまたはアプリケーションは、JDK 1.1 でも正しく実行できるはずです。うまく実行できないとすれば、潜在的なセキュリティに関する問題を修正したか、または実装や設計のバグを修正するために互換性を犠牲にしたごく少ない場合を除いて、それはバグです。詳細については、互換性の項を参照してください。

もちろん、 Java アプリケーションはブラウザでは実行できませんし、実行するには JDK または「Java ランタイム」だけが必要です。
