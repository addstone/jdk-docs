JavaSoft へのコンタクト方法

JavaSoft は、米国 Sun Microsystems, Inc. の １部門であり、JDK の開発を行っています。

テクニカルヘルプ

プログラミングまたはその他の技術的な質問がある場合は、JavaSoft 電子メールアドレスに記載の場所からテクニカルヘルプを得られます。

バグの提出

バグを提出する前に、「既知のバグ (Known Bugs)」レポートを確認してください。バグおよび機能の要求を提出するには、web ページの「バグの報告 (Reporting Bugs)」を使ってください。「修正済みバグ (Fixed Bugs)」のリストもあります。

コメントおよび提案

上記の同じ web ページ JavaSoft 電子メールアドレス に記載の JavaSoft の特定チーム宛てに電子メールを送ることもできます。 JDKの開発に関するコメントや提案は歓迎しますが、これらのチームアドレス宛てに、返答を要する個人的なプログラミングや技術的な質問を送信する目的には使用しないでください。 ただし、JDK 設計の改善に結びつくような質問は歓迎いたします。

その他のコメントや提案はすべて、 jdk-comments に送信してください。 コメントはすべて検討しますが、すべてに対しては個々に返答することはできません。

web ページの最後およびドキュメンテーション内でも、関連項目についての JavaSoft の担当者の電子メールアドレスがあります。どうぞお気軽に、担当エンジニアに直接コメントや提案を送ってください。

なお、バグの提出やコメントを JavaSoft へ電子メールで送る時は、英語で書いてください。

Web および FTP からのアクセス

JavaSoft には、World Wide Web または FTP サーバーからアクセスできます:
  • web からアクセスする場合は、http://java.sun.comを使います。
  • FTP ユーザは、名前には "anonymous" を、パスワードには自分の完全な電子メールアドレスを使い、"ftp.javasoft.com" に接続できます。例えば、電子メールアドレスが "john.doe@mycompany.com" の場合、次のようになります。

      iftp ftp.java.soft.com
      name: anonymous
      password: john.doe@mycompany.com
