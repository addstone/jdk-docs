Java(TM) ツール参照ページ (Windows)

基本ツール
javac
Java(tm) プログラミング言語で書かれたプログラムをバイトコードにコンパイルする Java 言語コンパイラ。

java
Java(tm) プログラミング言語で書かれたプログラムを実行する Java インタプリタ。

jdb
Java(tm) プログラム内のバグを見つけて修正するために有効な Java 言語デバッガ。

javah
Java(tm) クラスのための C ヘッダファイルと C スタブファイルを作成する。これらのファイルは、Java プログラミング言語で書かれたコードと、C などのその他の言語で書かれたコードを接続するのりのような役目をし、コードが相互に作用する。

javap
コンパイルされた Java(tm) ファイルを逆アセンブルして、Java バイトコードを表示させる。

javadoc
Java(tm) ソースコードから HTML 形式の API ドキュメンテーションを作成する。

appletviewer
World-Wide Web ブラウザを使わずにアプレットを実行させる。

RMI ツール
rmic
java.rmi.Remote インタフェースを実装する Java オブジェクトの、スタブクラスファイルおよびスケルトンクラスファイルを作成する。

rmiregistry
指定のポートでリモートオブジェクトレジストリを開始する。リモートオブジェクトレジストリは、RMI サーバが利用するブートストラップ名付けサービスである。

serialver
ひとつ以上のクラスのシリアルバージョン UID を返す。

国際化ツール
native2ascii
非 Unicode{tm} Latin-1 (ソースコードまたはプロパティ) ファイルを Unicode{tm} Latin-1 に変換する。

JAR ツール
jar
複数のファイルを 1 つの Java アーカイブ (JAR) ファイルに統合する。

デジタル署名ツール
javakey
アーカイブファイルのデジタル署名を生成し、エンティティ、キーおよび署名のデータベースを管理する。
