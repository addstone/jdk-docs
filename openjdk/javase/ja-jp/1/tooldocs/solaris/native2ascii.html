native2ascii - ネイティブ - ASCII コンバータ

ネイティブコード (Latin 1およびUnicode(tm)以外) のファイルを Unicode(tm)コードに変換します。

機能説明

native2ascii [options] [inputfile [outputfile]]

説明

Java コンパイラおよび Java ツールは、Latin-1 または Unicode(tm) コード(¥udddd 表記法) のファイルだけを処理することができます。native2asciiは、その他の文字コードのファイルを Latin-1 または Unicode(tm) コードのファイルに変換します。

outputfile を省略した場合、標準出力に出力されます。さらに、inputfile を省略した場合、標準入力から入力されます。

オプション

-reverse
反対の操作を行います。つまり、Latin-1文字 または Unicode(tm) コードのファイルをネイティブコードに変換します。

-encoding encoding_name
変換処理が使用するコードの名前を指定してください。デフォルトのコードは、システムプロパティの file.encoding から与えられます。encoding_name の文字列は、クラス CharacterEncoding の引数として渡されるものでなければなりません。
